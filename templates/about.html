About This Project

A Python & Flask Technical Showcase

Welcome to the blog! This full-stack application was built from scratch using Python and Flask. It serves as the capstone project for my backend development studies, showcasing my ability to integrate databases, authentication, and dynamic content.

The application implements RESTful routing, secure user registration using salted password hashing, and relationship-based database management (users, posts, and comments) via SQLAlchemy.

Note: This site is for demonstration purposes only.
